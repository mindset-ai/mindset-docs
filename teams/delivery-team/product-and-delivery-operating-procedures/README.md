### Document Purpose and Scope

These procedures support ISO 27001:2022 compliance (Control A.5.37) by documenting key engineering processes. All procedures reflect actual current practices at Mindset AI Ltd.

#### What's Covered

* How we develop and deploy code
* How we test and validate changes
* How we configure and manage infrastructure
* How we maintain security and respond to incidents

#### What's Not Covered

* Coding style guides (team preference, not security control)
* Development environment setup (individual choice)
* Troubleshooting guides (operational, not compliance-related)
